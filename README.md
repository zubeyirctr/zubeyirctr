# Zübeyir Çatar

İstanbul Arel Üniversitesi Bilgisayar Mühendisliği öğrencisiyim. Yapay zeka entegrasyonları, mobil uygulama geliştirme ve finansal teknolojiler (trade botları) üzerine çalışmalar yapıyorum.

### Yetkinlikler
- Programlama Dilleri: C, JavaScript, Java
- İlgi Alanları: Yapay Zeka, Mobil Uygulama Mimarileri, Algoritmik Ticaret

### Güncel Projeler
- Diet.ai: Yapay zeka destekli diyetisyen asistanı uygulaması.
- Trade Bot: Finansal piyasalar için geliştirilmiş otomatik işlem botu.
- Mobil Oyun Projesi: Hikaye tabanlı mobil oyun geliştirme süreci.
